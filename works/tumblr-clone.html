Tumblr Clone

Tumblr Clone is a simplified replica of the popular blogging platform, Tumblr.

About This Project And My Role

The goal of this project was to recreate the backend for an existing web application using backend technologies such as Express, PostgreSQL, knex.js etc. Although some HTML and CSS were used to make the app function, we tried keeping it simple for this project in order to focus more on the backend technologies.

My role was to create different post requests so that the user can post content onto the site. Tumblr has several different ways to post content using different mediums such as images, video, audio etc. However, for the purpose of this project I decided to recreate just the text post functionality- for both the regular text and quote post options. I also created a delete request so that you're able to delete posts as well. My other tasks included aiding in setting up our database by creating the migration files (schema changes) and seeds (test data) for the users, as well as rendering the list of "recommended" users on the right side of the page.

1. Create an account or log in using your credentials when you first land on the site.

2. Click on trash icon to delete posts.

3. Use search bar to search through posts.
