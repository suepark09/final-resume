Holy Grale Brewery

Holy Grale Brewery is a website for a fictional brewery based off the movie, Monty Python and the Holy Grail.

About This Project And My Role

This was our first group project for DigitalCrafts. The goals of this project were as follows:


  • Learn how to work on a singular site as a team
  • Familiarize ourselves with the git workflow
  • Learn how to read other people's code and merge pull requests
  • Learn how to resolve merge conflicts within either GitHub or the IDE.

My role was to create the homepage and the base components for the site, such as the navigation, footer, as well as the base template.
